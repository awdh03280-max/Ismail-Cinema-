- [useNativeDriver on web](native-driver-web.md) — use `ND` from src/utils/animation.ts (Platform.OS !== 'web') instead of hardcoded true; affects SplashScreen, HeroBanner, MovieDetailsScreen, PlayerScreen, WatchPartyScreen, FamilyModePinScreen.
