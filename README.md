# Ismail Cinema

A Netflix-style React Native Expo movie application with Arabic and English support.

## Features

- Animated splash screen
- Netflix-style home page
- Arabic and English support
- Movie search using OMDb API
- Favorites system
- Continue watching tracker
- Movie details page
- Dark premium UI
- Bottom navigation
- Responsive Android design

## Installation

```bash
git clone https://github.com/awdh03280-max/Ismail-Cinema-.git
cd ismail-cinema
npm install
```

## Running

```bash
npm start
npm run android
```

## Project Structure

```
src/
├── api/              # API integration
├── components/       # Reusable components
├── i18n/            # Internationalization
├── navigation/      # Navigation setup
├── screens/         # Screen components
└── storage/         # Local storage
```
